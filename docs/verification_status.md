- 环境和关键依赖版本。
- 完整启动命令。
- task YAML 与对象列表。
- checkpoint 名称及来源。
- 随机种子。
- TensorBoard 曲线和终端日志。
- 对实际现象的描述，而不是从单次运行推断总体性能。
